This website . . .

is designed to be a handy reference source for the basic HTML and CSS language elements and vocabulary. It is aimed at both the beginner and those more experienced who may need to refresh their memories. It does not teach you how to use HTML or CSS (except in the limited sense of how best to use the element refered to as part of refreshing the memory).

Having a basic understanding of both HTML and CSS is important for this reference material to be useful and we suggest finding a suitable course first (see available courses below). However, it may well be useful as an additional resource while you are doing such a course.

Notes on using the reference material:

BLUE text indicates the HTML or CSS reference material available.

RED text is code as it should be written in a text editor and is usually displayed in a fixed-width font. The particular focus of the code will be in a brighter red for emphasis.

BLACK text indicates particular instructions about using any of the elements or properties.

TO BEGIN AT THE BEGINING...

Before using this reference material, take a moment to understand six important terms: the HTML element, (and the element's possible attribute and its value); along with the three parts of a CSS Rule: the selector, the property, and its value.

THE HTML 'ELEMENT'

<p>...</p> opening tag | content | closing tag

The example above, in bold, is an HTML element. Like many HTML elements, it has an opening tag <p> and a closing tag </p>.

HTML elements can be further sub-divided, but don't worry about these just yet. As you progress, these sub-divisions will become more meaningful. They will help hone in your search for that elusive element name you just can't remember. For the moment, just click on the 'HTML Elements (Tags)' and scroll down to find the one you're looking for.

THE CSS RULE

selector {property: value;} e.g. 

<p>{padding: 10px;}</p>

The CSS rule is a little more complex. It has three parts to remember: the selector, property and value:

• The selector is often no more than an HTML element (without the angle brackets), as in the p example above;

• The property part of a CSS rule;

• The value part is usually easy to figure.
